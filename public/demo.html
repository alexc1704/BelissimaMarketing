Account Creation

Create your Belissima account

Users can create an account, enter their details, and begin using Belissima to discover beauty providers that fit their preferences.

Create a user account with basic personal details.
Store user preferences for a more personalized experience.
Access Belissima features after logging in.

Search & Filter

Find providers by what matters

Users can search and filter beauty providers by service type, location, price range, availability, and preferences.

Filter by services such as hair, nails, makeup, waxing, and brows.
Narrow results by location and provider details.
Make discovery faster than scrolling through scattered social media pages.

Point-Based AI Matching

AI scores providers by user request

Belissima uses a point-based AI recommendation system instead of a star ranking system. The AI evaluates provider details against the user's request and recommends providers with the strongest fit.

Points are assigned based on matching service type, location, budget, availability, and preferences.
The AI recommendation is tailored to the request the user makes.
Users receive ranked suggestions that support faster decision-making.

Provider Profiles

Review details before choosing

Each provider profile gives users helpful information such as services, pricing, contact links, policies, photos, location, and operating details.

View provider photos and business information.
Check service type, location, pricing, and policies.
Save favourite providers for later comparison.
